* <h2> Γλώσσα προγραμματισμού Grace: </h2>
* Φάση 1 (Λεκτικός - Συναντικός αναλυτής): </br>
        * Μεταγλώττιση προγράμματος: </br>
            * Χρησιμοποιήθηκε ο σκελετός της εργασίας για Java που υπάρχει 
        στο e-class. Αυτό σημαίνει πως όλα τα goals του maven είναι διαθέσιμα
        με την πρόσθεση του ότι η εντολή "mvn jar:jar", παράγει ένα εκτελέσιμο 
        jar του μεταγλωττιστή (target/compiler-1.0.jar) (προτείνεται η χρήση 
        της εντολήςmvn install για αυτόματη παραγωγή του αρχείου compiler-1.0.jar)
            * Η βασική κλάση (Main.java) βρίσκεται στο πακέτο 
            compilers.main (src/main/java/compiler/main/Main.java)
        * Εκτέλεση του προγράμματος:
            * Το πρόγραμμα δέχεται ένα αρχείο προς εκτέλεση σαν (το μοναδικό) όρισμα.
                Αν αυτό δεν παροχηθεί (ή το αρχείο που έχει δοθεί δεν υπάρχει), τότε
                δέχεται input από το standard input.
            * Το πρόγραμμα μπορεί να εκτελεστεί κανονικά (java path/to/Main),
                δίνεται ωστόσο η δυνατότητα αυτόματης εκτέλεσης, η οποία περιγράφεται
                παρακάτω:
                * Στον φάκελο tester, υπάρχει το python3 script tester.py
                  το οποίο δέχεται ως όρισματα τα παρακάτω:
                  1. Το path όπου βρίσκεται το εκτελέσιμο java
                  2. Το path όπου βρίσκονται τα .grace αρχεία
                  (Για περισσότερες πληροφορίες, μπορείτε να τρέξετε:
                    python3 tester.py --help)
                  και εκτελεί όλα τα αρχεία .grace μέσα στον φάκελο αυτόματα
                  (εκτυπώνοντας το output)
